Curl-to-c#

Instantly convert curl commands to C# code

This tool turns a Curl command into c# code. Currently, it knows the following options: -d/--data/--data-binary, -H/--header, -I/--head, -u/--user, compressed, ---url and -X/--request. There's probably bugs; please contribute on GitHub!

This script derives from John C's excellent curl-to-php.
